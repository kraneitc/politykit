# Charterfall

Charterfall is an experimental commercial game prototype being developed on top of PolityKit's open simulation framework.

This directory may include early product framing, prototype design notes, implementation plans, and integration notes for using PolityKit as the simulation layer for Charterfall. These materials are shared publicly for development transparency while the prototype is being explored.
